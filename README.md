### 擴充功能介紹:
將支援的活動網站頁面轉相關資訊加入到換到Google Calendar中，以便記錄與提醒。

### 目前支援網站:
1. Accupass活動
2. 年代售票系統
3. 易遊網
4. 可樂旅遊
